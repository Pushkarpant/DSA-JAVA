```java
package arrays;
import java.util.*;

/**
 * LeetCode #1 - Two Sum
 * Difficulty: Easy
 *
 * Problem:
 * Given array and target, return indices of two numbers
 * that add up to target.
 *
 * Approach: HashMap
 * For each number, check if (target - number) was seen before.
 *
 * Time: O(n) | Space: O(n)
 *
 * Key Learning:
 * HashMap converts O(n²) brute force to O(n).
 * Think "what complement do I need?" not "try all pairs".
 */
public class TwoSum {
    public int[] twoSum(int[] nums, int target) {
        HashMap<Integer, Integer> seen = new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            int complement = target - nums[i];

            if (seen.containsKey(complement)) {
                return new int[]{seen.get(complement), i};
            }

            seen.put(nums[i], i);
        }

        return new int[]{};
    }
}
```

```java

```

**Sliding Window** — Use when: subarray/substring of size k
```java

windowSum -= nums[i - k];
```

**HashMap Frequency** — Use when: counting, duplicates, anagrams
```java

```

**Stack Matching** — Use when: brackets, nested structures
```java

```

